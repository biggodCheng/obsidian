# 60-30-10法则

## 定义

**公众号内容配比策略：60%价值内容 + 30%专业内容 + 10%互动内容**

## 配比结构

### 60% 价值内容

**目的**: 建立信任，吸引目标用户

**选题范围**: 情感、职场、婚恋、热点，都可以写

**关键约束**: 落脚点必须回到你的目标用户身上

**错误写法**（纯八卦号）:
```
标题: 《XX和XX官宣离婚，男方净身出户》
正文: 讲离婚细节、财产分割、恋爱史
结尾: 所以啊，娱乐圈的爱情真的不能信
```

**正确写法**（价值内容）:
```
标题: 《XX离婚拿了2亿：女人有钱，底气就是不一样》
正文: 热点只讲20%，转到观点——
      "你发现没有，有钱的女人离婚，叫'重新开始'；
       没钱的女人离婚，叫'晚景凄凉'。
       不是说一定要离婚，而是说，女生一定要有自己的收入来源。"
结尾: 引导关注，或埋一个"我是怎么做到的"的钩子
```

**差异**: 读者不只吃了瓜，还记住了你的观点

### 30% 专业内容

**目的**: 让用户知道你是做什么的、能帮她解决什么问题

**核心原则**: 也在提供价值，只是落脚点指向产品

**常见误区**:
```
❌ 我的课程很好，快来买
❌ 限时优惠，不要错过
```

**正确写法**（好的30%）:
```
✅ "根据我们3年跟踪的500个案例，80%的用户都遇到了这个问题..."
✅ "上周有个学员，她的情况是...最后我们帮她解决了这个问题"
✅ "我总结了一个三步法，第一步...第二步...第三步..."
```

**核心区别**: 提供价值 → 落脚到产品

### 10% 互动/其他

**目的**: 社群维护、答疑、增加用户粘性

**形式**: 问答、社群互动、话题讨论等

## 三种内容的配合逻辑

```
60% 价值内容（引流）
    ↓
建立信任，让用户记住你的观点
    ↓
30% 专业内容（转化）
    ↓
展示专业能力，让用户知道你能帮她
    ↓
10% 互动内容（维护）
    ↓
增强关系，收集反馈
```

## 快速判断法

**怎么判断一篇内容属于60%还是30%？**

看标题和结尾：

| 内容类型 | 标题特征 | 结尾特征 |
|----------|----------|----------|
| 60% 价值 | 借热点/讲观点 | 引导关注、表达观点 |
| 30% 专业 | 讲方法/展示案例 | 引导咨询/了解产品 |

## 核心洞察

1. **不能变成纯八卦号**: 60%内容也要落脚到目标用户
2. **不能变成硬广**: 30%内容也要提供价值
3. **配比不是死的**: 根据账号阶段和用户反馈调整
