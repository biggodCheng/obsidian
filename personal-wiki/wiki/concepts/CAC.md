# CAC（Customer Acquisition Cost）

## 定义

**CAC（Customer Acquisition Cost）**即用户获取成本，是指获取一个新用户所需的平均成本。

---

## 计算公式

```
CAC = 总获客成本 / 新增用户数
```

**获客成本包括**:
- 广告投放费用
- 销售团队薪酬
- 营销工具费用
- KOL合作费用
- SEO/SEM费用

---

## 两种获客模式对比

| 模式 | 传统投放 | 病毒循环 |
|------|----------|----------|
| CAC | 高（数十到数百元） | 趋近于零 |
| 增长速度 | 线性增长 | 指数级增长 |
| 可持续性 | 依赖预算持续投入 | 自增长 |
| 典型渠道 | 广告/SEO/KOL | 用户自发分享 |

### SBTI的CAC

**成本构成**:
- 开发成本：一个HTML文件
- 托管成本：Cloudflare免费托管
- 推广成本：0
- **CAC ≈ 0**

---

## 病毒循环的CAC优势

### 传统模式 vs 病毒循环

**传统投放模式**:
```
投入广告费 → 获取用户 → 停止投放 → 增长停止
```

**病毒循环模式**:
```
初始用户 → 分享带来新用户 → 再分享 → 指数增长
     ↑                                      ↓
     └──────────────────────────────────────┘
```

### 数学原理

假设每个用户带来K个新用户（病毒系数）：
- K > 1：指数增长，CAC趋近于零
- K = 1：线性增长
- K < 1：逐渐衰减

---

## 降低CAC的方法

### 1. 产品即传播

**SBTI案例**:
- 27张人格图 = 27个传播炸弹
- 产品结果本身就是传播素材
- 用户分享不需要额外激励

### 2. 零阻力分享机制

**关键**:
- 分享是自然行为，不是额外任务
- 给用户预设好的分享话术
- 创造@好友的场景

### 3. 出海产品的零翻译成本

**情境依赖型内容**:
- 视觉冲击 + 情绪共鸣
- 翻译后几乎不丢味道
- "原来我们都是小丑" → "Turns out we're all clowns"

---

## CAC与LTV的关系

### 健康的商业模式

```
LTV（用户生命周期价值） > CAC（用户获取成本） > 0
```

**理想状态**:
- LTV / CAC > 3：健康
- LTV / CAC > 5：优秀
- CAC ≈ 0：病毒循环

### SBTI的变现潜力

虽然CAC≈0，但变现路径清晰：
- 广告变现：百万级PV
- 结果解锁付费：详细报告/配对分析
- API接口：给其他APP提供功能
- 品牌联名：特别适合出海
- 周边商品：T恤/贴纸/手机壳

---

## 核心洞察

> 对出海产品来说，CAC趋近于零意味着什么？意味着你的增长不再依赖预算，而是依赖产品的传播力。

降低CAC的核心：不是削减营销预算，而是让产品本身具备传播力，让每个用户都成为免费的推广渠道。
