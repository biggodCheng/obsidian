# 龙虾AI

## 摘要

基于 Obsidian + Claude Code 构建的个人知识管理工具集，包含 article-processor、ob-note、ob-search、ob-todo 等核心技能。

## 内容

龙虾AI是一套基于 Claude Code 构建的 Obsidian 知识管理工作流工具集，核心组件包括：

- **article-processor**: 文章处理技能，抓取 URL 内容并自动拆卡
- **ob-note**: 笔记管理技能，快速记录洞察和笔记卡片
- **ob-search**: 知识库搜索技能，支持关键词、标签、类型过滤
- **ob-todo**: 任务管理技能，记录和跟踪日常工作卡片

核心理念：将 AI 作为知识管理的"编译器"，一次摄入、持续维护。
