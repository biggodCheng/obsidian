# 情绪化流程设计法

## 方法内容

将传统的"数据收集型"流程改造为"情绪体验型"流程，提升用户记忆深度和分享欲望。

### 核心原则
**不是在收集数据，是在制造体验**

### 对比

| 维度 | 传统设计 | 情绪化设计 |
|------|----------|-----------|
| 题目/步骤风格 | 中性、客观、冷静 | 沙雕、挑衅、共鸣 |
| 用户反应 | "哦知道了" | "哈哈哈哈" / "这不就是我？" |
| 记忆效果 | 第二天忘了 | 记住一辈子 |
| 分享欲望 | 无 | 忍不住截图 |

### 四种情绪触发类型

1. **笑** - 沙雕文案让人笑（如："怎么样，被我拿捏了吧？"）
2. **愣** - 挑衅语气让人愣住（如："卧槽，我怎么是这个人格？"）
3. **不舒服** - 戳中痛点（如："嘴上说屎，手上默默打开Excel"）
4. **释怀** - 包装成有文化的梗（如：把"屌丝"包装成"第欧根尼传人"）

### SBTI的5维度情绪模型

| 模型 | 情绪类型 | 示例 |
|------|---------|------|
| 自我模型 | 自嘲/存在焦虑 | "屌丝独白"到"人生虚无主义" |
| 情感模型 | 情感共鸣 | "对象5小时不回消息" |
| 态度模型 | 价值观触动 | "被激素支配" |
| 行动驱力 | 行为共鸣 | "便秘30分钟怎么办" |
| 社交模型 | 社交共鸣 | "网友面基"到"电子围栏" |

### 隐藏彩蛋机制
SBTI设置了2道隐藏酒鬼检测题：爱好选"饮酒"+态度选"白酒当白开水喝"→强制分配DRUNK人格。
- 目的：给特定用户制造惊喜和分享欲望
- 启发：在流程中设置"彩蛋"，增加话题性

## 适用场景
- 测试/问卷类产品
- 用户onboarding流程
- 内容创作（文章/视频/课程）
- 任何需要"用户记住"的体验设计
