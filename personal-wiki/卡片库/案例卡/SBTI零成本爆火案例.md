# SBTI零成本爆火案例

## 案例事实

**产品**: SBTI人格测试
**前身**: B站UP主「蛆肉儿串儿」的免费网页
**技术栈**: 1个HTML文件 + 27张图片 + Cloudflare免费托管
**推广预算**: 零
**效果**: 2026年4月9日冲上热搜，服务器被挤爆，大量用户进不去页面

## 关键数据

| 维度 | 数据 |
|------|------|
| 技术成本 | 接近零（HTML+图片+免费托管） |
| 推广费用 | 零（无投放、无KOL、无广告） |
| 获客方式 | 纯病毒循环 |
| 用户获取成本(CAC) | 趋近于零 |
| 传播规模 | 热搜+服务器被挤爆 |
| 内容资产 | 27张人格图+30道题 |

## 成功因素拆解

### 因素1: 27张人格图 = 27个传播炸弹
- 每张图都是独立的转发理由
- 表情包+梗图+吐槽素材包
- 文案设计：挑衅+共鸣+自嘲

### 因素2: 30道情绪题 = 30次情绪过山车
- 不是收集数据，是制造体验
- 5个模型×6道题+2道隐藏彩蛋
- 情绪波动=记忆深度

### 因素3: 15维度评分系统 = 专业感+娱乐性
- 科学感和娱乐性双重满足
- 三层缓冲：乐子/分析/免责
- 不抗拒任何心态的用户

### 因素4: 天然出海基因
- 情境依赖型幽默，零翻译成本
- 已有英文镜像站运行
- 品牌联名在海外特别吃香

## 可复用的设计模式

1. **极简技术+丰富体验**: HTML+图片就能做到，不需要复杂后端
2. **结果即传播素材**: 产品使用结果本身就是社交货币
3. **隐藏彩蛋机制**: 制造惊喜和话题
4. **多层用户缓冲**: 让不同心态的用户都能找到舒服的位置

## 变现路径（尚未实现但清晰）

- 广告变现（百万级PV）
- 结果解锁付费
- API接口输出
- 品牌联名（出海重点）
- 周边商品
