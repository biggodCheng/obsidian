# 普通人启动三问

## 方法步骤

如果你觉得"我啥都不会，就是个普通上班族"，这才是大多数人的状态。没有显赫的title，没有硬核的技能。

但你的优势被严重低估了：**执行力。**

在想法泛滥的时代，能把一件事踏踏实实做完的人，已经赢了80%的人。

你不知道做什么？就问自己三个问题：

### 问题1：过去半年，别人反复问过你什么？

**为什么这个问题有效**：
- 那可能是你的知识盲区里的亮点
- 别人反复问 = 这是你无意识中擅长的
- 别人愿意问 = 这是有需求的

**示例**：
- "你怎么总能找到便宜的好东西？"
- "你是怎么把家里整理得这么干净的？"
- "你怎么总是知道最新的优惠信息？"

### 问题2：你顺手帮朋友解决过什么问题？

**为什么这个问题有效**：
- 那可能是你能提供的最小服务单元
- 对你来说顺手 = 对别人来说有价值
- 朋友愿意求助 = 这事儿你能做

**示例**：
- 帮朋友修过电脑 → 技术支持服务
- 帮朋友做过旅行攻略 → 行程规划服务
- 帮朋友选过保险产品 → 咨询服务

### 问题3：你因为兴趣，比身边多数人多知道点什么？

**为什么这个问题有效**：
- 兴趣驱动的知识是真实的积累
- 比身边人多知道一点 = 比市场平均多一点
- 哪怕是养多肉、淘二手书、找打折券

**示例**：
- 养多肉 → 多肉养护指南
- 淘二手书 → 二手书捡漏技巧
- 找打折券 → 优惠信息整合

## 普通人的变现方向

### 方向1：信息差搬运

**具体做法**：
- 你不是专家，但你可以成为"信息的整理者"
- 比如把市面上所有关于"小红书起号"的免费教程看一遍
- 去粗取精，整理成一份清晰的步骤清单
- 卖9.9元

**核心原则**：你赚的不是信息创造的钱，是信息筛选和重组的时间钱

### 方向2：资源整合

**具体做法**：
- 你认识一些做设计的朋友
- 也认识一些需要设计的小老板
- 你不需要会设计，你只需要当好这个"中间人"
- 赚取合理的介绍费或差价

**核心原则**：连接就是价值

### 方向3：轻量级服务

**具体做法**：
- 资料整理、行程规划、社群维护、打卡陪跑
- 这些不需要惊天动地的创意
- 只需要耐心和靠谱
- 从99元的社群，到999元的月度陪跑

**核心原则**：靠谱本身就是稀缺资源
