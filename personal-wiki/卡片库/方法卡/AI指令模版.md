# AI指令模版

## 方法

给AI的指令，必须包含你的判断。只有这样，AI的发散才有边界和方向，生成的内容才有你的骨架和灵魂。

## 指令模版

```
我的判断是【你的假设】。

请你围绕这个判断，完成以下工作：
第一，【具体任务1】
第二，【具体任务2】
第三，【具体任务3】
```

## 示例

**场景**：销售话术优化

**错误指令**（泛化，没有判断）：
```
请写一篇关于销售话术的文章，要干货，要落地，要通俗易懂。
```

**正确指令**（包含你的判断）：
```
我的判断是：我们的销售话术，在回答价格时，没有瞬间衔接价值，导致了客户的"价值断点"。

请你围绕这个判断，完成以下工作：
第一，列举在销售中客户询问价格时的5种常见场景；
第二，针对每种场景，分别生成3段强调"价值衔接"而非"价格解释"的应对话术草稿；
第三，找出2个因为价格沟通失败导致丢单的经典商业案例。
```

## 核心要点

### 1. 必须包含你的判断

指令的核心是"我的判断是【】"，这是你为AI设定的轨道和边界。

### 2. 围绕判断发散

AI不是随意发散，而是围绕你的判断发散：
- 找案例来支持你的判断
- 做阐述来解释你的判断
- 写反驳来强化你的判断

### 3. 具体明确的任务

不要让AI"写一篇文章"，而是给出具体任务：
- "列举5种常见场景"
- "生成3段话术草稿"
- "找出2个商业案例"

## 为什么这样有效

1. **AI的发散有了方向**：围绕你的判断，而不是随意发散
2. **内容有了骨架**：你的判断是脊椎骨，AI填充血肉
3. **保持你的控制权**：你始终掌控问题的定义权和价值的判断权
