# 商业闭环始于低价

## 判断内容

**你的第一个产品，就应该是一个小到你自己都不好意思收费的东西。**

目的不是为了赚这笔钱，而是为了完成一次完整的"商业闭环"：需求沟通 → 交付服务 → 获得反馈 → 收到报酬。

**判断标准**：
- 如果你觉得"这个价格是不是太低了" → 刚刚好
- 如果你觉得"这个价格合理" → 可能太高了
- 如果你觉得"我是不是在割韭菜" → 完美

**典型定价**：
- 99元：简历优化、账号诊断
- 199元：一次咨询、一次话术
- 299元：一次面试模拟、一次深度服务

**为什么必须低价启动**：
1. **降低成交门槛**：客户不需要冒大风险
2. **完成闭环**：验证你能从0到1赚到钱
3. **获得真实反馈**：付费客户的反馈比朋友真诚
4. **建立信心**：闭环带来的信心比看100本书都有用

**适用场景**：
- 第一次推出服务
- 进入新领域
- 没有成功案例之前

**不适用场景**：
- 已有成熟案例和口碑
- 高客单价的定制服务

**核心原则**：
先成交，再优化。在实战中迭代，远比在空想中规划来得有效。
