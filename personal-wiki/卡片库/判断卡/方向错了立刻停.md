# 方向错了立刻停

## 判断内容
跟 AI 来回三四轮还没解决？停下来重开对话。继续在错误方向追加，每一轮都在为之前所有错误上下文买单。重新开始反而更省。

## 判断依据
- Agent 滚雪球效应：每轮都带完整上下文，错误方向下边际成本递增
- 每一轮修正都带着前面所有对话的上下文成本
- Claude Code 中可用 Rewind（按两下 Esc）撤回上一轮，比 /clear 更精细
- Rewind 撤回后，后续每轮不再为那次错误买单

## 适用条件
- 与 AI 来回 3-4 轮仍未解决问题
- 发现自己在纠正 AI 的错误方向
- Agent 模式下长会话中方向偏离
