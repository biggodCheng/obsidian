# B2B养C端抗周期

## 判断内容

**B2B大额订单给底气做C端免费产品。经济好时企业预算大B2B好卖，经济差时个人用户省钱DIY工具好卖。这个组合是抗周期的。**

**判断标准**：
- 如果你的产品线全是C端 → 容易受经济周期影响
- 如果你有B2B能力但没有利用 → 错失了抗周期优势
- 如果你想降低单一产品的风险 → 考虑B2B+C端组合

**Steve的案例**：

**B2B端**：
- zwibbler：嵌入式白板SDK
- 定价：$5,999永久许可
- 一年20单就是$120K ARR
- 这是六个产品里最赚钱的

**C端**：
- rhymebrain：8语种押韵词典
- 定价：单用户$0（广告+API）
- 靠B2B订单支撑，敢做免费C端产品

**抗周期逻辑**：
- **经济好时**：企业预算大，B2B好卖
- **经济差时**：个人用户省钱，DIY工具好卖

**适用场景**：
- 有B2B销售能力或渠道
- 有C端产品或想法
- 想要平滑经济周期的独立开发者

**不适用场景**：
- 纯C端消费产品（如社交网络）
- 纯B2B企业服务（如SaaS）
- 不具备B2B销售能力

**核心原则**：
这个组合是抗周期的。大额B2B订单给你底气，让你敢做C端免费产品或低价策略。
